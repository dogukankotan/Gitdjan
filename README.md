

# Gitdjan
Web Interface to own git server with django

### Requirements
- Django==1.9.7
- libgit2
- pygit2

### Installation
- libgit2 installation:
 - ubuntu/debian: ``sudo apt-get install libgit2-dev``
 - arch: ``sudo pacman -S libgit2``
- ``sudo pip install -r requirements.txt``
