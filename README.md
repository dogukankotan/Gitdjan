# Gitdjan
Web Interface to own git server with django

#### Requirements
- Django
- libgit2
- pygit2

#### Installation
- libgit2 installation:
- - ubuntu/debian: ``apt-get install libgit2-dev``
- - arch: ``pacman -S libgit2``
- ``pip install -r requirements.txt``
